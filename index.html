Política de Privacidade

Esta aplicação utiliza as APIs da Meta Platforms, Inc. exclusivamente para coleta e análise de dados de desempenho de anúncios, de forma interna e restrita.

Não coletamos, armazenamos ou compartilhamos dados pessoais de usuários finais. As informações acessadas referem-se apenas a dados agregados de contas de anúncios previamente autorizadas.

Os dados obtidos são utilizados apenas para fins de relatórios, automações internas e análise de desempenho publicitário.

Nenhuma informação é vendida, repassada ou utilizada para fins comerciais externos.
